# LearnAndroid - Proyecto de Aprendizaje

## Descripción
Aplicación de práctica para aprender Android Studio y Kotlin desde cero.

## Tecnologías
- Kotlin 1.9.x
- Android SDK 34
- Jetpack Compose
- ViewModel + LiveData/StateFlow

## Estructura
- `ui/` - Composables y pantallas
- `viewmodel/` - ViewModels
- `data/` - Repositorios y modelos
- `util/` - Utilidades

## Conventions
- MVVM con Compose
- Nombres en camelCase
- Un archivo por clase/interface
- Inyección con Hilt
